# Moon Blaster

A simple 2D game made in Godot 4.x.

## Objective
Destroy the moon in the center of the screen within 60 seconds!
Choose from 3 difficulty levels.

## Controls
- **Arrow Keys / WASD**: Rotate and Move
- **Space**: Shoot

## How to Run
1. Open Godot 4.x.
2. Import this project (`project.godot`).
3. Press Play (or F5).

## Features
- Destructible Moon (using Geometry2D clipping).
- 3 Difficulty Levels affecting fire rate and damage.
- Win condition: Destroy 95% of the moon.
- Lose condition: Timer runs out.
